# Inday

### Prep

```shell
npm install
```

### Run Web app

```shell
npm run web
```

### Run Desktop app

```shell
npm run build:web
npm run desktop
```

### Run iOS app

```shell
npm run ios
```

### Run Android app

```shell
npm run android
```

### Run server

```shell
DATABASE_HOST=127.0.0.1 DATABASE_NAME=inday DATABASE_USERNAME=postgres DATABASE_PASSWORD=DATABASE_PORT=5432 LOGGER_LEVEL=debug PORT=4000 npm run server
```
