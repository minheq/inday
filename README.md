# Inday

### Prep

```shell
npm install
```

### Setup

- `index.js` is entry point for mobile (Android and iOS)
- `index.web.tsx` is entry point for web
-

For iOS

```
pod install
```

### Run Web app

```shell
npm run web
```

### Run Desktop app

```shell
npm run build:web
npm run desktop
```

### Run iOS app

```shell
npm run ios
```

### Run Android app

```shell
npm run android
```
